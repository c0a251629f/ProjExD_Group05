# 土管除けこうかとん

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 常に重力で落下し続けるキャラクター（こうかとん）をキー操作でバウンドさせ、迫りくる障害物を避け続けるエンドレスアクションゲーム

## ゲームの遊び方
* スペースキーを押下することで、こうかとんが上方向にバウンド（ジャンプ）する
* 画面右側から流れてくる上下の障害物（土管や壁など）の隙間をくぐり抜ける
* こうかとんが障害物に衝突する、または画面の上下端にはみ出してしまうとゲームオーバーとなる

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画
* 障害物の移動、時間経過の加速、スコアは時間経過
* 重力による落下とスペースキーによるジャンプ処理
* 障害物のランダム生成と左方向へのスクロール処理
* こうかとんと障害物の当たり判定

### 分担追加機能
* 障害物とリスタート機能の追加（田島）
* スコアを増やすコインのアイテム（高瀬）
* スタートまでのカウントダウン機能、ゲームオーバー画面と総合スコアの表示（池谷）
* 一定時間スコアを３倍にするアイテム（岡田）
* 一定時間透明になるアイテム（清藤）
